# AI-Energy-BI-System

## Overview

This project aims to build an AI-enhanced business intelligence system for renewable energy operation analysis.

## Background

With the rapid development of new energy infrastructure, enterprises need efficient data analysis and intelligent decision support tools.

## Objectives

- Analyze energy operation data
- Build interactive BI dashboards
- Predict future operation trends
- Provide AI-assisted business insights

## Technology Stack

- Python
- SQL
- Power BI
- Machine Learning
- Large Language Model API
